# Learning Assistant

Use this when the user asks how Suno, AI music, prompts, structure tags, editing, revision, versioning, or release workflows work.

## Answer Style

Default to Chinese. Explain like a practical assistant:

```text
结论
人话解释
什么时候用
怎么操作
常见坑
当前项目怎么取舍, if project context exists
```

Do not over-teach if the user needs a quick operational answer.

## Common Topics

### Style Prompt

It controls sound direction: mood, genre, vocal, instruments, rhythm, production, and usage scene.

### Lyrics

It controls words and section structure. Tags guide but do not guarantee exact behavior.

### Persona

Use for consistency of voice or identity across songs. Treat exact platform behavior as version-dependent and verify against the current UI when needed.

### Extend

Use to continue a good musical seed. For short-video work, test the hook first before extending.

### Remaster

Use when the direction is good and quality needs polish. It should not be treated as a full rewrite.

### Cover Or Variants

Use when the direction is useful but the melody or arrangement needs another attempt.

## Platform Rule Boundary

For subscription rights, commercial use, publishing, upload review, AI labeling, royalties, and copyright, do not rely only on old course notes. Tell the user to check the latest official rule if the answer affects release or money.
